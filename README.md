ИИ-Гид по Сортировке (EcoScanner Bot)
Суть: Бот помогает правильно утилизировать мусор по фотографии.
Как работает: Пользователь отправляет боту фото предмета (например, стаканчика от кофе или пластиковой бутылки).
Бот с помощью компьютерного зрения определяет тип материала и выдает инструкцию: можно ли это переработать, куда сдавать в вашем городе и как это влияет на уменьшение свалочного газа (метана).
